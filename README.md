# 🌟 BlogSphere — Social Blogging Platform

A modern full-stack blogging platform inspired by Instagram and Pinterest where users can create posts, upload images, like, comment, and interact socially.

---

# 🚀 Features

✅ User Authentication  
✅ Register & Login System  
✅ JWT Authentication  
✅ Create Blog Posts  
✅ Delete Posts  
✅ Like Posts  
✅ Comment System  
✅ Responsive Design  
✅ Pinterest/Instagram UI  
✅ Modern Glassmorphism Theme  
✅ MySQL Database Integration  
✅ REST API Backend  
✅ Mobile Responsive Layout  
✅ Smooth Animations  

---

# 🛠 Tech Stack

## Frontend
- HTML5
- CSS3
- JavaScript

## Backend
- Node.js
- Express.js

## Database
- MySQL

## Authentication
- JWT
- bcryptjs
